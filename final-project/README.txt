Description

MyForumApp is a full-stack web application that allows users to create accounts, manage profiles, and engage with a community by posting messages and replying to others. The app provides user authentication, a simple home feed with recent posts, and a profile view for each user.

The backend is built with Spring Boot and PostgreSQL, while the frontend uses React for a responsive user interface. The application includes JWT-based authentication and role management for users.

Features

User registration and login with hashed passwords

Create, view, and reply to posts

View user profiles

Home feed displaying recent posts

Protected routes requiring authentication

JWT-based token authentication

Technologies

Backend: Spring Boot, Spring Security, JWT, PostgreSQL

Frontend: React, React Router, Axios, CSS Modules

Database: PostgreSQL with JDBC for DAO operations
