# Module one final project - Banker Maintenance CLI

This is a basic command line application simulating an account maintenance program used by a banker. It utalizes file i/o to store and update data. A user is able to search for accounts using an account number and retrieve information including balances, signers, interest rates, etc. A user is also able to search for individuals by name or social security number and retrieve information such as birthday, address, accounts. etc. Users are able to make changes to this information.
